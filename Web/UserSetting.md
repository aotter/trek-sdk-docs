
# User Setting

> If you want to more attractive to lead ong user click Ad, you can send user's data let AotterTrek Ad system will just send user's interested Ad type.

#### Usage
```js
AotterTrek('setUser', {
  email: 'example@aotter.net',  
  birthday: '1999/01/01',
  fbId: '[USER_FACEBOOK_ID]',
  gender: 'MALE'  
});

AotterTrek('nativeAd', {
  selector: '#native-ad'
})
```
